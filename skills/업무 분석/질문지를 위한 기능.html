질문지를 위한 기능

설문지 시스템 드래그 앤 드롭 기능의 핵심 키워드

핵심 jQuery UI 컴포넌트
• jQuery UI Sortable: 드래그 앤 드롭 기능의 핵심 라이브러리
• handle: 드래그할 수 있는 요소 지정 (.re_wr_box_move)
• placeholder: 드래그 중 위치 표시자 (sort-placeholder)
• update 콜백: 드래그 완료 후 실행되는 함수

HTML/CSS 구조
• response_writing_area: 전체 질문 컨테이너 (Sortable 적용 대상)
• re_wr_box: 개별 질문 박스 (드래그 가능한 아이템)
• re_wr_box_move: 드래그 핸들 (마우스 오버 시 표시)

주요 기능 및 함수
• fn_addNewQuestionBox(type): 새 질문 박스 추가
• fn_initializeBox($box): 질문 박스 초기화
• clone(false): 이벤트 핸들러를 제외한 요소 복제
• 이벤트 위임: 컨테이너에 이벤트 핸들러 등록 ($container.on())

질문 유형
• 객관식(objective): 라디오 버튼 선택
• 주관식(subjective): 단답형 텍스트 입력
• 장문형(long): 여러 줄 텍스트 입력
• 체크박스(checkbox): 다중 선택
• 파일 업로드(file): 파일 첨부

데이터 관리
• deletedQuestList: 삭제된 질문 ID 보관 배열
• data-id 속성: 서버에 저장된 항목 식별자
• data-type 속성: 질문 유형 식별자

이벤트 처리
• mouseenter/mouseleave: 드래그 핸들 표시/숨김
• click 이벤트: 질문 유형 변경, 복제, 삭제
• sortable update: 드래그 앤 드롭 완료 후 처리

동적 UI 업데이트
• 템플릿 기반 HTML 생성: 질문 유형별 템플릿 생성
• DOM 요소 관리: append(), after(), remove() 메서드
• 스크롤 애니메이션: 새 질문으로 스크롤 ($('html, body').animate())

문제해결 전략
• 요소 복제 시 고유 ID 제거로 데이터 충돌 방지
• 이벤트 위임으로 동적 요소 이벤트 처리
• 드롭다운 옵션 동적 생성을 위한 초기화 함수
